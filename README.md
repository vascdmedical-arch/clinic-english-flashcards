# Clinic English Flashcards

クリニック英会話の暗記用フラッシュカードアプリです。

## Features

- Category-based cards
- Japanese to English tap-to-flip cards
- English speech synthesis
- Speech speed control from 0.5x to 1.5x
- Random mode
- Review-again marks
- Custom sentence additions saved in the browser

## Publish

This app is a static site. It can be published with GitHub Pages from the repository root.
